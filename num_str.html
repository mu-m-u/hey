數字的運算:

加( + )、減( - )、乘( * )

除法有兩種: 小數除法( / )、整數除法( // )

平方( a**b ): 代表a的b次方

取餘數( % )


字串的運算:

跳脫( "Hell\"o" ): 可以將雙引號印出來

字串的串接用加號( + )或空白( ) ex: "Hello" "World" = "HelloWorld"

換行用\n或三個引號( """string""" )把字串包起來 在三引號內換行會被印出來

重複字串用乘號( * ) ex: "Hello"*3 = "HelloHelloHello"

字串會對內部字元做編號(索引): 從0開始 ex:

s = "Hello"

# = 01234

s[0] = 第0號的H , s[2] = 第二號的l

s[1:4] = ell 從第一號e開始(包含)取到第四號o結束(不包含)

s[1:] = ello 從第一號e開始(包含)取到最後

s[:4] = Hell 從頭取到第四號o(不包含)結束
